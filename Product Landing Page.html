What are Project Grunt Boots?

The Project Grunt Boot was designed as a high ankle boot, with real-time terrain adjustment and full camouflage capabilities. From Common to Legendary, each boot carries a different maximum Gear level and perk potential.

Boot Levels

Each Boot has a different max level set to it. With the lowest maximum level being level 10 from a Common Boot and the max level being level 30 from a Legendary Boot.

Boosts and Perks

Lower class Boots will only have one perk attached to it, while more expensive and higher class boots like the Epic Boot has 2 perks attached and the Legendary Boot has 3 perks attached.
